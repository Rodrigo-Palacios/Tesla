## 🚀 Clone Tesla Project - Kickoff

The development of the **Clone Tesla** project has begun, the initial setup includes the creation of the navigation bar and the first touches of **background styling**.

The road to a sleek Tesla clone has started!

---

### 🛠️ Tech Stack

- **Astro** – Static site builder for lightning-fast performance
- **JavaScript** – Adds interactivity and dynamic behavior
- **Tailwind CSS** – Utility-first CSS framework for rapid UI development
